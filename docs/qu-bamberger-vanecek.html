Quadratwurzelverfahren

Quadratwurzelverfahren (“Square root method”) (78) is an alternative method for calculating SMOG (Bamberger-Vanecek). Where SMOG (Bamberger-Vanecek) uses a 30-sentence sample, Quadratwurzelverfahren uses a 100-word sample; otherwise, they produce the same results.

The Quadratwurzelverfahren Formula

G = √((C*(100/W))/(S*(100/W))*30) - 2

Where:

G Grade level
C Number of 3+ syllables (“Drei- und Mehrsilber”) words
S Number of sentences
W Number of words

Based on the examples from Bamberger-Vanecek (78), the grade-level score should be truncated to one-point precision, not rounded.
